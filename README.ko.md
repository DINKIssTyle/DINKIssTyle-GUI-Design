# DINKIssTyle GUI Design

DINKIssTyle GUI Design은 모던한 플랫 스타일의 GUI 레이아웃을 효율적으로 생성장할 수 있도록 설계된 강력한 드래그 앤 드롭 인터페이스 빌더입니다. [Wails](https://wails.io/)로 구축되어 고성능 Go 백엔드와 유연한 바닐라 자바스크립트 프론트엔드를 결합했습니다.

주요 목표는 생성형 AI 모델이 디자인된 레이아웃을 쉽게 이해하고 복제할 수 있도록 **AI 친화적인** 출력 구조를 제공하는 것입니다.

## 🚀 주요 기능

### 1. 윈도우 및 컴포넌트 아키텍처
- **윈도우 (루트)**: 반경, 배경 제어 및 AI 컨텍스트 설명을 지원하는 완전 맞춤형 루트 컨테이너입니다.
- **컴포넌트**: 버튼, 입력창, 스위치, 카드, 섹션, 탭, 테이블, 디바이더 등 풍부한 빌딩 블록 세트를 제공합니다.
- **드래그 앤 드롭**: 스마트 스냅 가이드와 자동 정렬 기능을 갖춘 직관적인 배치 시스템입니다.

### 2. 고급 컴포넌트 관리
- **속성 패널**: 위치, 크기, 색상, 글꼴, 스타일 등 모든 측면을 세밀하게 제어할 수 있습니다.
- **계층 시스템**: 컴포넌트를 중첩할 수 있습니다 (예: 카드 내부의 입력창, 탭 내부의 버튼).
- **탭 시스템**: 탭별 자식 가시성 제어 및 늘리기(Stretch) 정렬을 지원하는 고급 탭 컨트롤입니다.
- **테이블 편집기**: 테이블 셀 내용을 위한 내장 편집기입니다.
- **레이어 제어**: 단일 및 다중 선택 모드에서 맨 앞으로, 앞으로, 뒤로, 맨 뒤로 보내기 기능을 지원합니다.

### 3. AI 준비형 내보내기
- **XML 내보내기**: LLM 컨텍스트 주입을 위해 특별히 설계된 구조화된 `<Window>` / `<Components>` XML 파일을 생성합니다. 설명 및 계층 관계를 포함합니다.
- **JSON 내보내기**: 프로젝트 저장/불러오기를 위한 전체 상태 내보내기입니다.
- **동적 파일명**: 내보내기/저장 대화상자가 프로젝트 제목을 기반으로 파일명을 제안합니다.

### 4. 개발자 경험
- **히스토리**: 적절한 상태 관리를 갖춘 강력한 실행 취소/다시 실행 시스템입니다.
- **반응형 툴바**: 창이 좁을 때 스크롤 화살표가 나타나 모든 버튼에 접근할 수 있습니다.
- **네이티브 다이얼로그**: Wails 런타임을 통한 크로스 플랫폼 확인 대화상자 (macOS 호환).
- **사용자 정의 색상 팔레트**: 프로젝트별로 저장되는 사용자 정의 색상 프리셋입니다.
- **단축키**:
  - `Ctrl + Z`: 실행 취소
  - `Ctrl + Y`: 다시 실행
  - `Ctrl + D`: 선택 요소 복제
  - `Delete`: 선택 요소 삭제
  - `Ctrl + Scroll`: 확대/축소
  - 방향키: 선택된 요소 이동 (텍스트 입력 중 차단됨)
- **확대/축소**: 캔버스 줌 지원 (25% - 200%).

## 🛠 기술 스택
- **프론트엔드**: HTML5, CSS3, Vanilla JavaScript (무거운 프레임워크 없음)
- **백엔드**: Go (Golang)
- **프레임워크**: Wails v2

## 📦 빌드 및 실행
실시간 개발 모드로 실행하려면:
```bash
wails dev
```

배포용 빌드:
```bash
wails build
```

## 📋 최근 업데이트 (2026.01)

- 좌/우 화살표가 있는 반응형 툴바 스크롤 네비게이션 추가
- macOS 호환성을 위한 네이티브 확인 대화상자 구현
- 단일 선택 모드에서도 레이어 제어 기능 사용 가능

- 텍스트 입력 중 의도치 않은 요소 이동을 방지하기 위한 방향키 차단
